# v1.1

- 恢复接近 v1.0.6 的表格操作体验：每列有标题，表头可拖动调整列宽，保留左侧行选择栏。
- 增加“更新方式”列：自动识别、解压覆盖、静默安装。
- 支持 GitHub Release 安装包资产：`.exe`、`.msi`、`.msix`、`.msixbundle`，可后台静默安装。
- GitHub API 被限流时，自动改用网页解析 Release tag 和资产下载链接。
- 本地版本检测增强：优先读取主程序 exe 版本；目录填错时可按软件名、仓库名、Release 资产名在同级目录兜底查找主程序。
- 资产选择增强：按 Windows、x64/x86/ARM64、压缩包/安装包自动打分；x64 电脑会排除 ARM64/aarch64/armv8 资产。
- 说明文档补充“资产筛选(可选)”正则规则和常用示例。
